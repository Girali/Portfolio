Road editor
The road editor was my first tool made for unity editor in order to help the level designers to create faster and better level design for an arcade racing game named “Vega Missyl Racing Game”.

The principal is pretty simple, first I recreated a simple unity spline editor. It’s based on the Sebsatian Lague tutorial series about bezier curves.
Then I look up on a conference named “Unite 2015 - A coder's guide to spline-based procedural geometry” in order to learn the algorithm behind generating meshes at runtime and spline based geometries.
The logic behind is pretty simple, in order to create such a road editor, we need for our spline to have a UP and a RIGHT direction. Then we need to generate points on the spline that are evenly spaced (the point structure need to hold the up and right directions, so we can use it for the road).
The mesh logic is straight forward from this point, we generate slices of our road at places we created our previous points structure, finally we have to join the different road parts already generated the mesh looks continuous.
