Mesh morphing
A student project which aim to create a mesh morphing algorithm.
1 weeks
2 persons
Dec 2019
My approach to this problem was to first create an easy-to-use architecture for meshes in order to separate triangles of a mesh into structures that can be manipulated at will.
Then I try to morph a 2 mesh with equal numbers of triangles.
The next step was to try morphing from different triangles count, which was much more difficult. But basically I was duplicating certain triangles in the process or deleting some in order to create an illusion of morphing mesh.
The project was never polished in order to have a non dislocating morphing algorithm and a more elegant way of moving triangles.
