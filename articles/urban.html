Procedural Urbanization
Procedural Urbanization is a VR urban architecture tool made to create procedural cities on a terraformed landscape.

I was the project manager and the project main developer. I did not contribute to the code of the VR UI system and the save loader.
Here you can see a presentation video of the result achieve on the project.
At first, we have the terrain modification which can add, substrate, flatten or smooth the terrain, I integrated an erosion simulate in order to give the terrain a more realistic feeling.
When the user is done with the terrain, he can confirm and the textures of the terrain as well as the trees will be generated procedurally depending on their height and the stiffness of the terrain.
After this step, the user is free to place, delete or modify roads. The road segments that are placed generate additional elements based on the elements that are under and over them.
The user can as well place urban furniture on side walks. And complete his creation with building and parks that generate their own addition elements, based on the terrain shape.
