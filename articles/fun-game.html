F.U.N. Game is an old school type online FPS, based on a capture the flag game mode. The player incarnate one of the pre-determined classes and fight for the victory.

Beside the fact I never decided what the F.U.N. acronym means for this project, it was my first real networking experience.

MORE ABOUT THE PROTOTYPE
However, the acronym F.U.N. was used because the goal of the project was to provide the maximum fun potential out of the gameplay. The game was heavily inspired by Team Fortress 2 and the Tribes series.
This project, was all along designed to be a prototype with 1 game mode, and nice game mechanics.
It can sound very poor at the first view, but the game have many to offer.
The progress of this project can be seen on the following page, where I reported all changes and features added to the game throw my journey.
And as mentioned in the last update of the blog page, this prototype served me as a base to create my most complete named Vandalhalla.
