My approach
My approach
For the first part of the project, I need to generate a 2D version of the landscape in order to create an appealing and complex island. I will not explain the entire process and the difference parts in detail, but resume what I have done.
This 2D version helps to get the rough shape of the island, because it has all the data we need like height, humidity, river generated based on height, lakes, …
Then for the 3D part, I use the 2D data in order to generate the mesh. But due to software limitation we can not generate infinite triangle meshes in Unity, in addition the collision of the physic engine can’t handle this type of complexity.
So in order to create this massive landscape, I divide the entire structure into chunks that are meshes with lesser triangles.
To finish, I add some colours to the mesh, the project had some upgrades like trees and rocks that I placed procedurally or overall shape modification at a micro level, so the landscape get more complex when at a player height.
