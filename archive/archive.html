F.U.N. GAME [BLOG]

It’s a project of a game which aim the goal to provide the maximum fun potential of the gameplay. It’s a multiplayer game inspired by Team Fortress 2 and the Tribes series.

With this project, I’m aiming only to get a very solid prototype game with one game mode, and solid game mechanics.

It can sound very poor at the first view, but the game have many to offer.

This blog will serve me as a motivation goal and I will edit and post update on the ongoing project.

Update 10 (03/05) Classes:

This week, I made the 4 main classes playable in the game.

The first to be made is the Scout, which I wanted to be the fast one.

Each class of the game have one main weapon, one secondary, a gravity gun and C4 packs (unlimited amount but only 3 can be placed at once, one movement ability and one special ability (can be everything)).

The Scout :

Primary : Shotgun /// Secondary : Light Machine Gun /// Movement : Double Jump /// Special : Speed Up

The Medic :

Primary : Heavy Machine Gun/// Secondary : Defibrillator /// Movement : Slide /// Special : Medi-kit

The Engineer:

Primary : Small Machine Gun /// Secondary : Pistol /// Movement : Jetpack /// Special : Rocket Launcher

The Sniper:

Primary : Sniper /// Secondary : Revolver /// Movement : Gargling /// Special : Flash Grenade

I included some new way for the player to move with more fluency, of course the different ways will not be accessible for one player at once. It will be dispatched on different classes like in Team Fortress 2 (like the scout and his double jump).

Update 3 (07/04) Car :

I think every game that is fun to play, there are vehicles that the player can interact with in every way possible. For this reason, the vehicle will interact with bullet shoots and explosions.

Update 2 (30/03) Explosion Mechanics :

Every explosion in the game will repulse the player from the explosion point. It can help the player to reach higher places or to grant him more speed.
